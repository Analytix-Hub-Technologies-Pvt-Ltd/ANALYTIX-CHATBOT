WordPress Integration Guide

To embed this elegant AI chatbot on your AnalytixHub WordPress site, follow these simple steps:

  1. Log in to your WordPress Dashboard.
  2. Go to Plugins > Add New, search for "Header and Footer Code Manager" (or any header/footer injection plugin) and install it.
  3. Create a new snippet, choose "Footer" as the location, set scope to "Entire Site", and paste the code block below.
  4. Click Save. The chat launcher bubble will instantly appear on your website!
